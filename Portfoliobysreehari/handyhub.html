HandyHub - Local Service Booking App

Personal MVP Project

Developed a full-featured mobile application that connects users with local service providers such as electricians and plumbers.

  • Implemented OTP-based authentication using Firebase Authentication for secure user login.
  • Built a real-time booking system using Cloud Firestore, enabling users to schedule services and track booking status.
  • Designed a provider dashboard allowing service providers to accept or reject booking requests.
  • Developed a modern, responsive UI in Flutter with smooth navigation and real-time updates.
  • Implemented role-based access for different User and Provider workflows.
